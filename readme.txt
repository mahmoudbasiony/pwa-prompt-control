== Installation ==

= From your WordPress Dashboard =

1. Visit “Plugins” → Add New
2. Search for Reviewshake Widgets
3. Activate Reviewshake Widgets from your Plugins page.

= From WordPress.org =

1. Download Reviewshake Widgets
2. Upload the “reviewshake-widgets” directory to your “/wp-content/plugins/” directory, using ftp, sftp, scp etc.
3. Activate Reviewshake Widgets from your Plugins page.
